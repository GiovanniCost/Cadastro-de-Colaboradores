
# 👥 Sistema de Cadastro de Colaboradores

Uma aplicação completa (Full Stack) desenvolvida para gerenciar o registro de funcionários de uma empresa. O sistema permite cadastrar o nome, idade, cargo e e-mail corporativo de cada colaborador de forma rápida e intuitiva.

Este projeto foi desenvolvido como evolução de uma base de estudos do **DevClub**.

---

## 🚀 Tecnologias Utilizadas

### Frontend
* **React.js** (Construção da interface dinâmica)
* **Axios** (Consumo da API REST e gerenciamento de requisições HTTP)
* **Styled Components** (Estilização avançada com o conceito de CSS-in-JS)

### Backend
* **Node.js** (Ambiente de execução JavaScript)
* **Express** (Framework para construção da API REST)
* **Mongoose** (Modelagem de dados e integração com o banco)
* **MongoDB** (Banco de dados NoSQL para armazenamento dos dados)

---


## 📋 Funcionalidades Principais
* **Cadastro completo:** Coleta de dados como Nome, Idade, Cargo e E-mail Corporativo.
* **Comunicação Otimizada:** Integração assíncrona entre o frontend e o backend através do Axios.
* **Estilização Moderna:** Interface limpa, responsiva e componentizada utilizando Styled Components.
